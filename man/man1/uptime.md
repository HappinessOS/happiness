## Name

uptime - Tell how long the system has been running

## Synopsis

```console
$ uptime
```

## Examples

```console
$ uptime
 18:22:01 up 1 min

$ uptime
 18:22:01 up 59 min

$ uptime
 18:22:01 up  2:01

$ uptime
 18:22:01 up 1 day, 0 min

$ uptime
 18:22:01 up 1 day,  1:00

$ uptime
 18:22:01 up 25 days, 21:30
```
